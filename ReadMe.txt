AMIGUI.mlappinstall is a MATLAB App Installer File that is used to run the software within MATLAB.

AMIGUI_Installer_web.exe is stand-alone application that is used to run the software without MATLAB.  (Note however that the MATLAB Compiler Runtime is still required.  The user will be prompted to download the MCR during installation).

The *.m files are the MATLAB source code used to generate the executables.
